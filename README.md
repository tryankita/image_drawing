# BINARY.SRC

A premium, canvas-based image processing tool that converts photos into high-contrast digital art formats.

![Banner](index.html) <!-- Placeholder representing the visual project identity -->

## Features

- **Multi-Filter Engine**:
  - **Binary**: Clean high-contrast black and white abstraction.
  - **Matrix**: Cyber-hacker aesthetic with green contrast manipulation.
  - **Pixel**: Digital block-style processing inspired by low-res displays.
- **Dynamic Controls**: Real-time threshold/intensity slider for fine-tuning output.
- **Theme Engine**: Persistent Dark/Light mode toggle with "Digital Paper" vs "Cyber Green" skins.
- **Export Result**: One-click PNG download for generated art.
- **Premium UI**: Glassmorphism panels, scanline overlays, and responsive design.
- **Mobile Optimized**: Fully responsive with touch-friendly controls and hamburger menu navigation.
- **Camera Support**:
  - **Live Stream**: Real-time webcam processing with performance optimization.
  - **Native Capture**: Direct integration with mobile device cameras for high-quality shots.

## Technology Stack

- **Canvas API**: High-performance pixel manipulation.
- **Vanilla JavaScript**: Lightweight and fast logic processing.
- **CSS3**: Modern glassmorphism, flexbox/grid layout, and smooth transitions.
- **Google Fonts**: Inter/Outfit/Monospace typography.

## Quick Start

1. Clone the repository.
2. Open `index.html` in any modern web browser.
3. Drag an image onto the processor or click **Upload Image**.
4. Adjust filters and download your result!
